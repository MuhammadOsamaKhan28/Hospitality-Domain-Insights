# Hospitality-Domain-Insights

Hotel Revenue & Performance Analytics Dashboard (Power BI)

Project Overview:
Developed an end-to-end interactive Hotel Performance Analytics Dashboard using Power BI to analyze revenue, bookings, occupancy, cancellations, and customer ratings across multiple cities and properties.

🔹 Key Highlights

Built a multi-page dashboard (Executive View & KPI View) for management-level insights.

Designed a star schema data model with fact and dimension tables.

Created 25+ DAX measures including:

Revenue, RevPAR, ADR, Occupancy %

Realization %, Cancellation %

WoW Change %

DSNR, DBRN, DURN

Implemented dynamic filters (slicers) for:

Property

City

Booking Platform

Week Number

Status

Performed week-over-week performance analysis for revenue, occupancy, and ratings.

Built visualizations including:

KPI Cards

Line Charts (Trend Analysis)

Donut Charts (Room Class & Booking Category)

Stacked Column Charts (Cancellation % by City)

Treemap (Bookings by Platform)

Matrix Table (Property-level metrics)

🔹 Business Insights Generated
      
      Identified $1.71B total revenue with 135K+ bookings.
      
      Analyzed performance differences between Weekday vs Weekend bookings.
      
      Compared revenue by Room Class (Elite, Premium, Presidential, Standard).
      
      Evaluated City-wise revenue contribution (Mumbai, Bangalore, Hyderabad, Delhi).
      
      Tracked Occupancy % and Average Ratings trends across weeks.
      
      Measured cancellation impact by month and city.
